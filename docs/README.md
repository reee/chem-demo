# 化学教学演示项目

这是一个用于展示化学教学演示项目的网站，托管在 GitHub Pages 上。

## 项目说明

本网站集合了各种用于中学化学教学的交互式演示项目，旨在帮助学生更直观地理解化学原理。

## 当前项目

- **化学反应速率影响因素模拟**：展示了浓度、压强和温度如何影响化学反应速率的交互式模拟。

## 技术栈

- HTML5
- CSS3
- JavaScript
- SVG

## 本地开发

如需在本地查看网站：

1. 克隆此仓库
2. 打开 `docs/index.html` 文件在浏览器中查看

## 部署

本网站使用 GitHub Pages 进行部署，从 `docs` 目录提供服务。
