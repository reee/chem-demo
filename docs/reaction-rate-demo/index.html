原理说明

一、模拟内容

本模拟展示了理想气体在发生反应时，浓度、压强和温度对化学反应速率的影响。通过调节不同的参数，可以观察到这些因素如何影响分子的运动和碰撞。

二、数据说明

• 相对浓度：与初始状态相比的浓度值（与分子数量正相关，与容器体积反相关）

• 相对压强：与初始状态相比的压强值（与浓度正相关，与温度的关系由查理定律确定）

• 活化分子百分数：具有足够能量进行反应的分子所占百分比（由阿伦尼乌斯方程的简化模型进行计算）

• 单位体积内活化分子数：每单位体积内的活化分子数量（由活化分子数和容器体积计算）

• 微粒平均速率：与初始状态相比的分子平均运动速率（根据麦克斯韦-玻尔兹曼分布进行计算）

• 碰撞频率：每5秒内发生的总碰撞次数（实时统计）

• 有效碰撞频率：每5秒内发生的活化分子之间的碰撞次数（实时统计）

三、局限性

• 为了方便统计，把所有活化分子的碰撞均记为了有效碰撞。
